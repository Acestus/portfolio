1. The Internet Parallel

Every transformative computing platform follows the same pattern: an open protocol enables interoperability, which enables an ecosystem, which enables network effects. The internet had TCP/IP → HTTP → HTML → Browsers. The agentic web has TCP/IP → MCP → Tool Schemas → Agents.

TCP/IP→ HTTP→ HTML/CSS→ Browsers→ The Web
TCP/IP→ MCP→ Tool Schemas→ OpenClaw→ Agentic Web

2. What Is MCP?

The Model Context Protocol, introduced by Anthropic in late 2024, is a JSON-RPC-based protocol that standardizes how AI agents discover and invoke external tools. It's analogous to HTTP in that it defines the verbs (requests), nouns (resources), and handshake patterns that allow any agent to talk to any tool server.

"MCP is HTTP for AI agents — a shared protocol that makes everything composable." — Peter Steinberger

Key properties:

  • Transport-agnostic — works over HTTPS, SSE, WebSocket, or stdio
  • Schema-driven — tools self-describe their inputs/outputs using JSON Schema
  • Bidirectional — agents can push context back to servers (sampling, roots)
  • Composable — connect multiple tool servers to one agent, or one server to many agents

3. Peter Steinberger & OpenClaw

Peter Steinberger — founder of PSPDFKit, veteran open-source developer — created OpenClaw as an open-source agentic AI agent. In early 2026 he joined OpenAI while insisting OpenClaw remain open-source under a non-profit foundation.

OpenClaw is to MCP what Mosaic was to HTTP. It's the first general-purpose agent that:

  • Runs locally — your data stays on your machine
  • Works with any LLM — Claude, GPT, Gemini, local models
  • Uses MCP natively — discovers and composes tool servers
  • Has a skills system — extensible via markdown skill files
  • Is fully open-source — no vendor lock-in, community governed
"The lobster is loose, and it's not going back into the tank." — Peter Steinberger

5. Why This Matters for Platform Engineers

As a platform engineer working with Azure, IaC, and Fabric, the agentic web isn't theoretical — it's practical:

  • MCP tool servers for Azure — agents can manage infrastructure via Bicep, query logs via KQL, manage policies
  • Fabric + agents — MCP-enabled agents can trigger pipelines, run KQL queries, monitor data quality
  • Policy governance via agents — automated compliance checks, remediation workflows, exemption management
  • Platform self-service — agents as the interface to platform engineering consoles

6. Open Standards Win

The internet wasn't built by one company. TCP/IP, HTTP, HTML, and CSS are all open standards that anyone can implement. The result was the largest collaborative system in human history.

The agentic web follows the same pattern: MCP is an open specification, OpenClaw is open-source, and the foundation governing it is non-profit. When protocols are open, ecosystems flourish.
